# 尿酸记录器 Uric Acid Tracker

一个极简的尿酸记录 H5 页面，支持数据本地存储和趋势图展示。

## 功能

- ✅ 记录尿酸值（μmol/L）
- ✅ 7天/30天/90天趋势图
- ✅ 数据统计（平均值/最高值/最低值）
- ✅ 历史记录管理
- ✅ 数据导出（CSV）
- ✅ 本地存储（LocalStorage）

## 使用

直接在手机浏览器打开即可使用。

## 技术

- HTML5 + CSS3 + Vanilla JS
- Chart.js 图表库
- LocalStorage 数据存储

## 参考范围

- 正常范围：208-428 μmol/L（男性）
- 低于 208：偏低
- 高于 428：偏高
